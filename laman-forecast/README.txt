
Forecast Page (simple modular)
------------------------------
Files:
- forecast.html        (open this file in the browser)
- css/style.css
- css/forecast.css
- js/api.js
- js/map.js
- js/charts.js
- js/ui.js
- js/main.js
- assets/icons/*.svg
- data/sample.json    (example response)

Notes:
- The page uses Open-Meteo (no API key) and RainViewer for radar tiles.
- For production, consider proxying requests if using API keys (OpenWeather).
- Open forecast.html in a browser (internet required for CDN and tiles).
